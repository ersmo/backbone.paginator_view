# Backbone.PaginatorView

Bootstrap style paginator backbone view

## Usage

```js
var view = new Backbone.PaginatorView({
  el: $('.paginator'),
  collection: new Backbone.Collection()
  lang: 'cn'
});
```

## Options

```coffee
defaults:
  lang: 'en'
```
